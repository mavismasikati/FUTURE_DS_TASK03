# 🛒 E-commerce Customer Behavior & Conversion Analysis

## 📊 Project Overview
This project analyzes **24,000+ e-commerce user sessions** to identify key drop-off points in the conversion funnel.  
The goal is to uncover actionable insights to improve conversion rates and optimize marketing strategies.

---

## 🎯 Objectives
- Analyze user behavior across the funnel (View → Cart → Purchase)
- Identify major drop-off points
- Segment performance by **price range, brand, and time of day**
- Provide data-driven business recommendations

---

## 💡 Key Insights
- 📉 **97% drop-off** between *View → Add to Cart*
- ⏰ Peak user activity occurs at **1:00 PM**
- 💰 Premium products ($500–$1,000) show stronger conversion once added to cart

---

## 🚀 Business Recommendations
- Improve product pages with **reviews, ratings, and trust signals**
- Run **marketing campaigns during peak hours (1 PM)**
- Use **remarketing strategies** for high-value product viewers

---

The project includes:
- Conversion Funnel Analysis
- Time-of-Day Traffic Trends
- Conversion by Price Range
- Brand Performance Analysis
